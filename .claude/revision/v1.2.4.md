# AT Pro v1.2.4 — Release Notes

**versionCode:** 26 | **versionName:** v1.2.4

---

## Thay đổi chính

### 1. Demo nuôi acc mở rộng — X, Instagram, Threads, Snapchat
- **AutomationEngine**: Thêm 4 `ServiceMode` mới: `X_NURTURE`, `INSTAGRAM_NURTURE`, `THREADS_NURTURE`, `SNAPCHAT_NURTURE`
- Thêm 4 public start methods (`startXNurture`, `startInstagramNurture`, `startThreadsNurture`, `startSnapchatNurture`) cùng 4 private session runners tương ứng
- Mỗi runner theo pattern: mở app → lướt feed → ngẫu nhiên tương tác (like/follow/repost) → đóng app
- **AppConstants**: Thêm `X_PKG`, `INSTAGRAM_PKG`, `THREADS_PKG`, `SNAPCHAT_PKG`

### 2. Lưu ý Demo (`DemoNoticeBanner`)
- Banner cảnh báo màu Amber xuất hiện đầu nhóm "Demo nuôi tài khoản khác" trong ServicesScreen
- Nhắc nhở: app tương ứng phải được cài trước, Accessibility Service phải bật

### 3. UI chuẩn Android (`ServicesScreen`)
- Palette mở rộng: màu thương hiệu chính xác cho X (`#14171A`), Instagram (gradient `#833AB4→#FD1D1D→#F77737`), Threads (`#101010`), Snapchat (`#FFFC00`)
- `PlatformDemoCard` — template card có thể tái sử dụng: gradient background, DEMO badge, chips, OpenServiceRow
- Thứ tự cards: Facebook → X → Instagram → Threads → Snapchat

### 4. Cài đặt cho từng nền tảng (`ConfigScreen`)
- Section mới "Demo nuôi" trong sidebar (group riêng)
- Slider riêng cho từng platform: thời gian nuôi, xác suất like, xác suất follow/repost/story view
- Fields lưu vào SharedPreferences, đọc qua `LocalRepository.loadFarmConfig()` → `AutomationEngine`

### 5. Thống kê theo nền tảng (`StatsScreen`)
- Platform tab row: TikTok / Golike / Facebook / X / Instagram / Threads / Snapchat
- **TikTok**: giữ nguyên thống kê cũ + range filter
- **Golike**: card số dư (khả dụng / đang giữ / chờ duyệt), danh sách TikTok accounts, tổng jobs
- **Demo platforms** (Facebook/X/IG/Threads/Snap): placeholder "Chế độ Demo — chưa theo dõi"
- Nhận `GolikeViewModel` qua parameter; truyền từ `MainScreen`

### 6. Tài khoản theo nền tảng (`AccountsScreen`)
- Platform tab row: TikTok / Facebook / X / Instagram / Threads / Snapchat
- TikTok: danh sách từ Room DB (giữ nguyên)
- Demo platforms: placeholder "tự động phát hiện khi chạy phiên nuôi"

### 7. Dashboard xử lý 4 mode mới
- `DashboardUiState.isDemoMode`: `true` cho tất cả 5 demo modes (Facebook + 4 mới)
- Start button phân nhánh `when(serviceMode)` cho đủ 7 case với màu accent riêng
- Farm mode toggle và Golike card ẩn khi `isDemoMode == true`

### 8. Version bump
- `build.gradle`: `versionCode 26`, `versionName "v1.2.4"`
- `build.yml`: `default: '1.2.4'`, `VER="1.2.4"`

## Lưu ý triển khai

- Demo modes chỉ cần app target được cài trên thiết bị — không cần đăng nhập Golike
- `isDemoMode` là computed val trên `DashboardUiState`, accessible từ UI layer
- `GolikeViewModel` được inject vào `StatsScreen` qua optional parameter (null-safe)
- `PlatformDemoCard` có thể tái sử dụng cho platforms mới trong tương lai
